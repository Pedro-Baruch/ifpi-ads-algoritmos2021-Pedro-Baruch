1. O que é, e para que é?

a. Pensamento Computacional
    O Pensamento Computacional é um processo que é composto por três estágios: a abstração, a automação e a análise. 
    Esse processo compreende a forma como problemas são resolvidos e expressar as soluções de forma 
    extremamente clara para que outras pessoas ou computadores
    entendam e possam repeti-la.
b. Algoritmo
    Algoritmos são descrições de processos e podem ser definidos usando diferentes tipos de linguagens
c. Programação
    É Descrever um algoritmo em uma linguagem que uma máquina é capaz de entender, como javascript, python, c++ e etc
d. Modularização com uso de Funções
    É uma forma de melhorar a organização e deixar o código menor, utilizando o recurso de exportar uma função de um arquivo
    para outro.
